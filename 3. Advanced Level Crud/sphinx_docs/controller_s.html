controller_s module¶

Uso de módulos de ‘tkinter’.

Para este módulo, desde ‘tkinter’ se realiza la importación de la vista del servidor de ‘view_s.py’.

controller_s.py:

Módulo que realiza la comunicación con el archivo ‘view_s.py’ del servidor. Contiene la clase Controller y llama a la clase Panel del módulo vista que crea la ventana principal de encendido y apagado del servidor.

class controller_s.Controller(root)[source]¶

Bases: object

Se define la clase ‘Controller’ y se utiliza el método ‘__init__’ para iniciar los atributos de los objetos creados.

Parameters

root – base de la interfaz gráfica (raíz, contenedor base de los widgets que forman la interfaz).
